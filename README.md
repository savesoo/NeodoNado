# NeodoNado
JAVA Spring framework, MySQL, JSP, javascript, REST API를 이용한 페이지 제작

- spring framework 사용
- jsp를 활용한 view 페이지 구성
- Mapper 이용한 CRUD 처리
- my batis 기반
- xml 이용하여 bean 생성하고 controller에 매핑
- Hikari CP
- DB 관리 : mySQL
- REST API 사용한 비동기통신
- bootstrap으로 UI 부분 구현

-----------------

# 구성

- user(회원)
  - 회원 가입
  - 로그인
  - 로그아웃
  - 마이페이지에서 정보 조회
    - 작성한 글 / 댓글 확인
  - 자동 로그인 기능
- board
  - list
  - 글 내용 보기
  - 글 작성
  - 수정
  - 삭제
  - list 더보기
  - comment
    - 댓글 작성
    - 수정
    - 삭제
    - 댓글 리스트 더보기
